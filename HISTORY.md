## 2026-04-29 14:00 — 프로젝트 관리 개념 설계

**작업 내용**: 발명신고 작성기에 "프로젝트" 개념 도입. 초기 프롬프트 문서를 업데이트.

**결정 사항**:
- 실제 발명신고 작성 세션의 단위 = **프로젝트** (발명 1건 = 프로젝트 1개)
- 새 프로젝트 생성 시: 이름 입력 UI → 해당 이름의 로컬 폴더 자동 생성 → 세션 시작
- 모든 중간/최종 결과물은 프로젝트 이름의 폴더에 저장
- 프로젝트 불러오기: OS 네이티브 **폴더 열기 다이얼로그**로 로컬 디스크 탐색

**저장 구조 설계**:

```
<프로젝트명>/
├── project.json               # 메타 + 워크플로우 상태
├── shared-document.json       # 공유 작업 문서 최신본
├── patent-document.json       # 발명신고서 최신본
├── conversation-log.json      # 에이전트 전체 대화 로그
├── patent-search-cache.json   # 특허 검색 결과 캐시
├── snapshots/                 # 라운드별 자동 스냅샷
├── exports/                   # 사용자 내보내기 파일
└── human-edits.json           # Human 편집 이력
```

**기술 선택**: 브라우저 환경에서 로컬 폴더 접근은 **File System Access API** (`window.showDirectoryPicker()`) 사용. Electron 전환 시 Node.js `fs` 모듈로 교체. 두 환경을 추상화하는 `fileSystemAdapter.ts` 레이어 설계.

**변경 파일**: `doc/prompt/patent-multiagent-claude-code-prompt.md`

## 2026-04-29 13:00 — 초기 프롬프트 문서 작성

**작업 내용**: 멀티 에이전트 특허 발명신고서 자동화 시스템 전체 설계를 문서화.

**핵심 설계 원칙**:

1. **에이전트 순차 실행** — 병렬 실행 절대 금지. 단일 진행 포인터 유지.
2. **논쟁 기반 생성** — Agent 1(Developer)과 Agent 2(Expander)가 수용/거부를 반복하며 청구항 완성.
3. **적대적 심사관** — Agent 3(Examiner)는 친절한 리뷰어가 아니라 공격적 거절 전문가.
4. **2단계 특허 검색** — 개념 분해 → 쿼리 생성 → 1차 검색 → 용어 추출 → 2차 정밀 검색.
5. **Human-in-the-loop** — 매 에이전트 턴마다 개입 가능. 게이트는 2개(초안 승인, 최종 승인).

**에이전트 구성**:

| 에이전트 | 역할 | 색상 |
|----------|------|------|
| Agent 0 (Structurer) | 자유 형식 아이디어 → 구조화된 Anchor Document | 보라 |
| Agent 1 (Developer) | 청구항 중심 설계 + 오리지널 범위 수호 | 파랑 |
| Agent 2 (Expander) | 발명 가능성 극대화 제안 (Agent 1 승인 필수) | 초록 |
| Agent 3 (Examiner) | 실제 특허 검색 기반 공격적 심사 | 주황/빨강 |

**워크플로우**:

```
Phase 0: 아이디어 입력
Phase 1: Agent 0 구조화 → Anchor Document 생성
Phase 2: Agent 1 ↔ Agent 2 피드백 루프 (최대 N라운드)
Phase 3: Agent 3 특허 검색 + 심사 → Agent 1 반박 루프
Phase 4: 최종 발명신고서 생성 + 저장 + 다운로드
```

**UI 레이아웃** — 3패널 구조:

```
┌──────────────┬──────────────────────┬───────────────────────┐
│  에이전트     │   공유 작업 문서      │   발명신고서           │
│  대화 패널   │   (편집 가능)         │   (실시간 업데이트)    │
└──────────────┴──────────────────────┴───────────────────────┘
│  Human 액션 바 (승인 / 수정 요청 / 일시정지)                  │
```

**생성 파일**: `doc/prompt/patent-multiagent-claude-code-prompt.md`
